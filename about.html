About Me

And what this site is for

I am John, and this is my site. I started this somewhere along my Python journey in order to document and share things I may have found interesting or hard, in the hope to help other people. Python is a fantastic language to learn, especially for beginners, due to its simple nature and logical approach to coding. It has expanded greatly in the last few years so now is a great time to start to learn.

Despite some percieved simplicity Python is extremely powerful. Companies like Netflix, Spotify and Instagram all use Python in some way for their websites. Not only is it endorsed by huge companies like those mentioned it is also excellent for those of us who are not programmers but need tasks automating, data processing or right down to an administrator bulk renaming files, or collating several reports into one.

It truely is a great time for anyone to start to learn programming and Python, and hopefully some content on this site will help you if you choose to start that journey.
